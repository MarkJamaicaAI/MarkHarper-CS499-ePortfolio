# Professional Self-Assessment

Throughout the Computer Science program at SNHU, I have developed a strong foundation in software development, algorithms, database systems, and secure application design. The projects included in this ePortfolio represent my growth from writing functional code to designing structured, secure, and scalable systems. Each artifact demonstrates progressive improvement in architectural organization, security implementation, and problem-solving ability.

One of the most significant areas of growth has been in secure development practices. Earlier projects focused primarily on functionality. However, through enhancement work, I implemented JSON Web Token (JWT) authentication, password hashing, environment variable configuration, and removal of hardcoded credentials. These improvements reflect my understanding of industry-standard security practices and responsible software development.

My work with algorithms and data structures strengthened my analytical thinking and optimization skills. Enhancing reinforcement learning logic and improving training stability required deeper understanding of algorithm efficiency and state-based reasoning. Similarly, database enhancements required implementing full CRUD operations, structured query validation, and secure authentication handling using MongoDB and PyMongo.

This portfolio demonstrates my ability to design, enhance, and secure full-stack applications while applying best practices in modular design, authentication, database management, and algorithm optimization. As I move forward in my professional career, I am confident in my ability to build maintainable, secure, and scalable software systems aligned with industry standards.
